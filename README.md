# IPM-DTI

IPM-DTI is a Multi-modal DTI prediction framework driven by Interaction Patterns. It integrates heterogeneous data—including protein sequences, molecular graphs, and explicit knowledge graph (KG) structures—to identify underlying drug-target interactions through learned interaction patterns. The framework first constructs a DTI KG from public databases. Multimodal encoders then map drugs and targets into a unified latent space to extract cross-modal features. To capture diverse interaction patterns, IPM-DTI decomposes the global KG into semantic subspaces based on these features, enabling focused local modeling. An RL-based reasoning module with a heuristic transition matrix guides exploration across subspaces to uncover latent associations. Finally, a fusion module combines structure- and RL-derived results for comprehensive DTI prediction. Experiments show that IPM-DTI outperforms single-modal and single-embedding baselines, demonstrating the effectiveness of our framework.
